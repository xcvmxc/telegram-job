# Источники Telegram для сканирования
#
# Ниже два раздела: каналы под "## Активные" сканируются; каналы под
# "## Неактивные" сохраняются, но пропускаются. Переносите строку между
# разделами, чтобы приостановить или возобновить канал, не удаляя его.
#
# Перечисляйте каждый канал простым буллетом, по одному в строке:
#     - @channelname
# Обычная строка без "- " тоже работает. Строки с "#" в начале игнорируются.
#
# ---------------------------------------------------------------------------
# ДОПУСТИМЫЕ ФОРМАТЫ (ставьте любой после "- ")
# ---------------------------------------------------------------------------
#   @channelname                 публичный канал или группа (username)
#   https://t.me/channelname     публичная ссылка — то же, что @channelname
#   https://t.me/+AbC123xyz      ПРИВАТНАЯ инвайт-ссылка (сначала вступите в Telegram)
#   -1001234567890               числовой id (приватный канал без username)
#
# Вы должны СОСТОЯТЬ в каждом канале из списка. Посмотреть все каналы/группы,
# где состоит ваш аккаунт — с готовым к вставке значением для каждого:
#   uv run --with telethon python ~/.tgjobs/telegram/tg_scan.py list

## Активные
# Удалите примеры ниже и добавьте свои каналы здесь, по одному буллету в строке.
# - @example_jobs_channel
# - https://t.me/another_public_jobs_channel
# - https://t.me/+ExamplePrivateInviteHash
# - -1001234567890


## Неактивные
# Каналы здесь НЕ сканируются. Перенесите строку вверх, в "## Активные", чтобы возобновить.
# - @some_channel_on_hold
